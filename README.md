🍽️ Zomato Front Page Clone

A responsive Zomato Front Page Clone built using HTML, CSS, and JavaScript. This project recreates the modern landing page design of Zomato with a hero section, search bar, navigation menu, and restaurant cards.

🚀 Live Demo

🔗 Live Demo:  https://github.com/Ganeshyadav77/zomatofrontpage.git

✨ Features
Modern Zomato-inspired UI
Responsive design for mobile and desktop
Hero section with background image
Restaurant search bar
Restaurant cards with images
Clean and user-friendly layout
Built with pure HTML, CSS, and JavaScript

HTML5
CSS3
